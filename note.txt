code ở trong máy | staging area | local repository | remote repositoty | 

local repository: nơi chứa code và các thông tin của mã nguồn khi sử dụng GIT ở trong máy tính của bạn
remote repository: nơi chứa code và các thông tin của mã nguồn khi sử dụng GIT ở trên tài khoản github của bạn

git init: tạo local repository
git add <tên-file>: đẩy các thay đổi của <tên-file> vào staging area
git commit: đưa các thay đổi từ staging area vào local repository, chụp lại snapshot tại thời điểm đó
git remote: kết nối local repository với remote repository
git push: đẩy các thay đổi từ local lên remote
